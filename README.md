#  ShopEase - E-Commerce Web Application

ShopEase is a modern and responsive e-commerce web interface designed to simulate an online shopping experience.  
It allows users to browse products, view product details, add items to the cart, and complete a simulated checkout process.

This project focuses on building a clean and interactive frontend using modern web technologies.

---

## 🚀 Features

- Product listing with responsive grid layout
- Product detail modal with description and rating
- Category filtering
- Price range filtering
- Product search functionality
- Sorting products by price and newest
- Add to cart functionality
- Shopping cart sidebar
- Quantity adjustment in cart
- Remove items from cart
- Checkout confirmation modal
- Login and Signup UI
- Notification toast messages
- Responsive design for mobile and desktop
- Local storage cart persistence
